## Installation

For breakdown of Mineshift:

```
shards build mineshift[1-6] && ./bin/mineshift[1-6]

```

For compiling your sandbox project
```
shards build sandbox && ./bin/sandbox

```

## Usage

Open a web browser and navigate to `localhost`.
